Easily set WooCommerce's 'Ship to a different address' checkbox default to checked or unchecked on the checkout page.

== Description ==

This plugin allows you to set the default state of the "Ship to a different address" checkbox on the WooCommerce checkout page.  
- By default, the checkbox will be **unchecked**.  
- To revert to WooCommerce default behavior, simply deactivate the plugin.  

No settings page is required — activate and go.

== Installation ==

1. **From the WordPress Admin Panel:**  
   - Go to **Plugins → Add New**.  
   - Search for "*Ship to a Different Address Checked/Unchecked for WooCommerce*".  
   - Click **Install Now** and then **Activate**.

2. **Manually via Upload:**  
   - Upload the `ship-to-a-different-address-checked-unchecked` folder to `/wp-content/plugins/`.  
   - Activate the plugin through the **Plugins** menu in WordPress.

== Frequently Asked Questions ==

= How do I use this plugin? =

Activate the plugin, and the "Ship to a different address" checkbox will be unchecked by default on the WooCommerce checkout page. Deactivate the plugin to revert.

= Does this plugin require any configuration? =

No. The plugin works immediately after activation.
